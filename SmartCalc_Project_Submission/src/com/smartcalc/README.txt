ABOUT:
SmartCalc is a futuristic calculator made in Java Swing.
It includes:
• Basic arithmetic (+, −, ×, ÷)
• Trigonometric functions (sin, cos, tan)
• Power, square root, inverse
• Neon blue futuristic UI
• Launcher screen with SmartCalc branding

HOW TO RUN:
1️⃣ Open terminal in project folder
2️⃣ Run:  java -jar SmartCalc.jar
OR
Double-click SmartCalc.jar if Java is installed
